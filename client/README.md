# Introducción

Este proyecto ha sido iniciado con [Create React App], herramienta oficial para arrancar aplicaciones React.

---

## Scripts Disponibles

En el directorio del proyecto, puedes ejecutar los siguientes comandos:

### `npm start`

Inicia la aplicación en modo desarrollo.  
Abre [http://localhost:3000](http://localhost:3000) en tu navegador para visualizarla.  

- La página se recargará automáticamente al guardar cambios en el código.  
- Cualquier error se mostrará en la consola.

---

### `npm test`

Ejecuta el entorno de pruebas en modo interactivo “watch”.  

---

### `npm run build`

Genera una versión optimizada para producción en la carpeta `build`.  
- Empaqueta React en modo producción.  
- Minifica el código y añade hashes a los nombres de archivo para una correcta gestión de caché.  
¡Tu aplicación quedará lista para desplegarse!
